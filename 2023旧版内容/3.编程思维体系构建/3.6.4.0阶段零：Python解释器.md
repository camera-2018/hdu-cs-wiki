你可以在终端与解释器进行交互

在终端中输入 python 即可进入解释器

解释器可以进行简单的表达式和语句操作

你可以自己把玩一下

```python
>>> 1 + 2
3
```

```python
>>> 3 - 2
1
```

```python
>>> 5 * 6
30
```

```python
>>> 7 / 4
1.75
```

```python
>>> 7 // 4
1
```

```python
>>> 7 % 4
3
```

```python
>>> 4 ** 3
64
```

同时可以输入 `exit``()` 或按 Ctrl+D 退出交互
