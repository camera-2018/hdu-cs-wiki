# 课程安排与时间表

## 课程安排原则

本课程前两周所有学生共同学习HTML、CSS、JavaScript及Git的基础知识，确保学生具备扎实的前端开发基础和版本控制工具的使用能力。第3周开始，课程将分为前端组和后端组，分别学习各自的技术栈。课程每周包含三部分：理论讲解（Doc）、实验操作（Lab）、独立作业（Homework）。

课程的设计宗旨是理论与实践结合，通过动手实验巩固知识，并最终完成一个完整的前后端项目。
